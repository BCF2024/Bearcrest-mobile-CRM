BEARCREST LOAN CRM VERSION 5.2 — OPERATIONS CENTER

WORKING CALENDAR
- Monthly CRM calendar
- Automatic loan follow-ups
- Automatic target closing dates
- Automatic term-sheet expirations
- Custom events tied to a loan
- Add an event from inside a loan
- Optional Google Calendar sync
- Double-click a calendar day to add an event

WORKING TASK CENTER
- Add a task globally or from inside a loan
- Due date, priority, assignee, category, notes, and status
- Open, completed, overdue, and due-today views
- Search and priority filters
- Display tasks on the CRM calendar
- Dashboard task and follow-up preview

INSTALL
1. Extract to:
   G:\My Drive\BearCrest_Loan_CRM_Version_5_2_Operations
2. Open index.html.
3. Temporary login:
   Username: admin
   Password: BearCrest2026!
4. Create your password.
5. Open Settings and add the existing Apps Script URL.
6. Select BearCrest CRM Documents.
7. Test and save.
