UPLOAD TO GITHUB
1. Open your existing repository.
2. Click Add file > Upload files.
3. Drag all files from this folder into GitHub.
4. Replace files with the same names.
5. Commit directly to the main branch.
6. Wait 1–3 minutes, then refresh your existing GitHub Pages URL.

IMPORTANT
This version automatically changes layout for desktop and phone. Loan records are still stored in each browser until the shared cloud database is connected. Use Export/Import for backups.
