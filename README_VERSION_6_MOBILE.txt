BEARCREST MOBILE CRM — VERSION 6.0

Version 6.0 is the phone-first companion to the BearCrest desktop CRM.

NEW MOBILE FEATURES
- Dedicated mobile home screen
- Bottom navigation designed for Samsung Fold and Android phones
- New Loan, Smart Sync, Scan Document, and Contacts quick actions
- Today / attention list
- Recent loans with one-tap call, text, and property map
- Global borrower/property/loan search
- Field Mode for calling, texting, directions, and quick notes
- Camera/document picker shortcut
- Full Version 5.4 CRM remains available underneath
- Google Drive shared database support remains included

HOW TO TEST ON YOUR PHONE WITHOUT HOSTING
1. Download the ZIP to your phone.
2. Use Samsung My Files to extract it.
3. Open index.html in Chrome or Samsung Internet.
4. Local-file browser restrictions may limit Google sign-in, Gmail, Drive, installation, and automatic sync.
