BEARCREST CRM VERSION 5.4 — MOBILE APP / PWA

This is an installable phone version of the BearCrest CRM.

PHONE FEATURES
- Mobile navigation drawer
- Phone-sized dashboard
- Full-screen loan editing
- Smart Sync
- Gmail and email attachments
- Google Drive documents
- Contacts
- Reusable Document Library
- Tasks
- Calendar
- Lenders
- Archive
- Branded documents
- Add to Home Screen / Install App support
- Offline caching of the app interface

SHARED PHONE + DESKTOP DATA
Version 5.4 adds a shared CRM database stored in:
BearCrest CRM Documents / BearCrest_CRM_Shared_Database.json

Use:
- Upload This Device to save this device's database to Google Drive.
- Download Cloud Data to load the shared database.
- Automatic Cloud Sync can upload changes every 2, 5, 10, or 30 minutes.

IMPORTANT
A local file opened with file:/// cannot be installed as a phone app.
The folder must be hosted on your website or another HTTPS web host.

GOOGLE APPS SCRIPT
Replace Code.gs with BearCrest_Cloud_Connector.gs from this package.
Save and deploy a New version of the existing web app.

FIRST SETUP
1. On the desktop CRM, connect Apps Script and click Upload This Device.
2. Host the Version 5.4 folder on your website using HTTPS.
3. Open the hosted address in Chrome on the Samsung phone.
4. Sign in and connect the same Apps Script URL.
5. Click Download Cloud Data.
6. Open Chrome menu and choose Install app or Add to Home screen.
